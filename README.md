# ShowroomKortrijk

Interactieve showroombouwer voor de showroom in Kortrijk.

## Bestanden

- `index.html`: interactieve tool met leeg grondplan, productkolom, verslepen, draaien, kopieren, verwijderen en PNG-export.
- `styles.css`: styling voor de showroombouwer.
- `app.js`: productdata en alle interactie voor de showroombouwer.
- `showroom-indeling-preview.png`: preview van de statische indeling.
- `showroom-bouwer-preview.png`: preview van de showroombouwer.

## Gebruik

Open `index.html` in een browser. Plaats producten vanuit de rechterkolom op het grondplan. De indeling wordt lokaal in de browser bewaard.
