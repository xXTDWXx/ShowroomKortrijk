# ShowroomKortrijk

Interactieve showroombouwer voor de showroom in Kortrijk.

## Bestanden

- `index.html`: opent automatisch de showroombouwer.
- `showroom-bouwer.html`: interactieve tool met leeg grondplan, productkolom, verslepen, draaien, kopieren, verwijderen en PNG-export.
- `showroom-indeling.html`: statische voorbeeldindeling.
- `showroom-indeling-preview.png`: preview van de statische indeling.
- `showroom-bouwer-preview.png`: preview van de showroombouwer.

## Gebruik

Open `index.html` of `showroom-bouwer.html` in een browser. Plaats producten vanuit de rechterkolom op het grondplan. De indeling wordt lokaal in de browser bewaard.
